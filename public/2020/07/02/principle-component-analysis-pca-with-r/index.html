Principle Component Analysis (PCA) with R

2020-07-02

Abstract

PCA is a statistical technique used in many different settings. Often it is used in data analytics to reduce the dimensions of a variable set. This is done through the linear combination of the raw variables to produce a set of components the aim of which to capture the maximum amount of variance with the minimum number of components.

\[PC_{1} = w_{1}x_{1} + w_{2}x_{2} + ... + w_{n}x_{n}\] \[PC_{2} = w_{n + 1}x_{n + 1} + w_{n + 2}x_{n + 2} + ... + w_{m}x_{m}\]
